# mkPrompts

Sammlung und Umsetzungshilfe für eine moderne Prompt-Bibliothek mit öffentlicher Ansicht und Admin-Bereich.

## Enthalten

- Laufender Prototyp (Python-Backend + Web-UI) im Repository.
- Zusätzlicher **Master-Prompt** für Claude/ChatGPT/Cursor:
  - `PROMPT_CHATGPT_CLAUDE_CURSOR.md`

## Nutzung des Master-Prompts

1. Öffne `PROMPT_CHATGPT_CLAUDE_CURSOR.md`.
2. Ergänze bei Bedarf deine echte Initialdaten-Liste.
3. Kopiere den Inhalt in dein bevorzugtes Coding-Modell.
